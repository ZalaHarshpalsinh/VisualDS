new Pointer( pointee, initialIndex, label )

Description
This class represents a visual array index variable. DO NOT create an object of this using the constructor. Always use getPointer() method of vArray to get an object of this.
Parameters
Name Type Description
pointee vArray The vArray object this Pointer object will point to
initialIndex number The initial index
label string The label to show below this pointer. Defaults to empty string.
Examples

Creating a pointer

//creating an array
			let arr = new vArray([1, 2, 3, 4, 5], 'My Array')
			
			// DO NOT CREATE POINTER LIKE THIS
			let pointer = new Pointer(arr, 0, "Incorrect Use") //Incorrect
			 
			//creating a pointer using the array
			let pointer = arr.getPointer(0, 'My Pointer')

move( change )

Description
Moves the pointer by the specified amount of change in index value.
Parameters
Name Type Description
change number The change to add in the index. Positive moves it ahead, negative moves it back. You can change the index however you want: no exception is thrown. Instead, to handle such cases, the drawIndex value is capped till one before start and one after the end of array.
Examples

Moving the pointer

//creating an array
			let arr = new vArray([1, 2, 3, 4, 5], 'My Array')
			
			//get the pointer
			let i = arr.getPointer(0, "Moving Pointer")
			
			// move forward
			i.move(4)
			
			// move backward
			i.move(-3)
